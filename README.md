# Discord-Bot-for-Member-Management
Member registration and management discord bot
Discord Bot for Member Registration and Management
This bot is designed to manage member registration and listing in Discord servers.

Commands
!register [name]: Registers a new user.
!list: Displays the list of registered members.
!permission: Displays the list of members with permissions.
!delete: Removes a member from the list.
Installation
Invite the bot to your Discord server.
Replace YOUR_BOT_TOKEN with the actual token.
Run the bot.
